# Sobre_mim

Me chamo Daniel Costa Carvalho e estou cursando Bacharelado em Sistemas de Informação na FTC.
Atualmente estou focado no ecossistema JavaScript, especialmente em Node, React e React Native.

JavaScript foi a linguagem com que aprendi a programar, porém não foi amor a primeira vista, por isso procurei por outras linguagens e me apaixonei pelo Python. Na nova linguagem, me interessei por me aprofundar um pouco mais e, para isso, desenvolvi algumas aplicações desktop principalmente com a biblioteca Tk. Na faculdade aprendi Java e, apesar de respeitar muito a linguagem, ainda continuei a desenvolver meus projetos pessoais com Python. Durante a faculdade também fiz algumas atividades em linguagem C. Minha visão do JavaScript mudou quando eu estava fazendo um trabalho da disciplina de programação para dispositivos móveis em grupo que tinha entre os membros meu amigo [Vinícios Grama](https://github.com/vinigr), ele tomou a frente de desenvolver o aplicativo com React Native.
